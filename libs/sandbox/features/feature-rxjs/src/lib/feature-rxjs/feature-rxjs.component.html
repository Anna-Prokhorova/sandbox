CUSTOM RXJS OPERATORS
BufferDynamic

Оператор, который буферизует значения исходного потока в массивы, которые динамически изменяются по размеру на основе второго потока-условия. Второй поток передавает значения, которые определяют текущий размер буфера. Как только буфер достигает указанного размера, массив передается далее по потоку, и буфер начинается с нуля. Оператор корректно обрабатывает ситуацию, когда поток-условие изменяет размер буфера в середине его заполнения.

DistinctUntilKeyChangedDeep

Оператор, который отслеживает изменения в сложных объектах, переданных в поток, и выдавает значение только тогда, когда изменяется определенный ключ в глубоко вложенной структуре объекта. Оператор должен принимать путь к ключу в виде строки (например, 'user.address.city') и игнорировать объекты с теми же значениями в этом ключе. Оператор должен быть устойчив к вложенности объектов и массивов, а также корректно обрабатывать изменения внутри массивов.

ForkJoinWithTimeout

Оператор, который работает аналогично forkJoin, но с добавлением тайм-аута для каждого из потоков. Если один из потоков не завершился в течение заданного времени, он должен быть заменен на значение по умолчанию.
